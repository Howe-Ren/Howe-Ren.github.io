NewBing注册
  • 准备工作：1. 魔法工具（VPN）。 2. Microsoft Edge Dev浏览器。 3. 微软账号。 4. 申请NewBing资格。
  • 主要操作：
    1. 下载浏览器
      • 下载链接：Download Microsoft Edge Insider Channels
      • 点击“Download”下载即可
      • 之后界面选择浏览器的语言，可以“English”或列表底部的“简体中文”
    2. 登录微软账号
      • 打开安装好的浏览器，点击左上角头像进入登录界面
      • 填写个人账号信息
    3. 申请NewBing
      • 成功登录后，进入申请网页链接：The New Bing - Learn More
      • 点击“加入等待列表”或“Join the waitlist”（我这边已经加入，所以中间是“ChatNow”）
    4. 打开NewBing新世界
      • 点击蓝色“b”气泡图标，展开侧边栏，输入即可开始对话！
      • 也可在浏览器搜索框搜索，然后 鼠标滚轮上翻/触摸屏上翻 进入BingChat
